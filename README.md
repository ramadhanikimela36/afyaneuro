# AfyaNeuro — Neurosurgery, Explained

A medical education platform breaking down neurosurgery and neuroanatomy for students, junior doctors, and curious minds across East Africa and beyond.

Built by **Ramadhani Kimela**, a medical intern on the path to neurosurgery.

## 🧠 About

AfyaNeuro makes neurosurgery and neuroanatomy accessible through clear, practical articles written from real ward experience. Each piece covers:

- **Neuroanatomy** — Understanding brain and spine structure
- **Clinical cases** — Real scenarios from the ward
- **Exam prep** — Practical guides for medical students
- **Career insights** — The path to neurosurgery from East Africa

## 📖 Features

- Responsive, dark-themed design with neural pathway animations
- Article grid with metadata (date, read time)
- About section with stats
- Support/donation integration
- Mobile-friendly navigation
- Social media links

## 📝 Adding Articles

To add a new article, edit `index.html` and add an entry to the `ARTICLES` array:

```javascript
{
  topic: "Neuroanatomy",
  title: "Article Title Here",
  summary: "Brief description of the article.",
  date: "Month Year",
  readTime: "X min read"
}
```

Add new articles at the **top** of the array for them to appear first.

## 🛠 Customization

- **Colors**: Edit CSS variables in `<style>` (`:root` section)
- **Fonts**: Google Fonts imported in `<style>`
- **Content**: Update text in HTML sections or JavaScript arrays
